Privacy Policy — stel-token-refresh

Effective date: 2026-05-18

stel-token-refresh ("the extension") is a browser tool that retrieves and displays authentication tokens for the user's own accounts.

Data the extension accesses

  • The stel_token cookie set by oauth.telegram.org.
  • The access_token cookie set by app.send.tg.

These values are read from the browser's cookie store and shown in the extension's popup so that the user can copy them.

How the extension uses data

  • Token values are displayed in the popup window only.
  • To obtain a fresh access_token, the extension sends requests to oauth.telegram.org and app.send.tg — the same authentication flow the user would otherwise perform manually. Requests go only to those two services.

What the extension does NOT do

  • It does not transmit any data to the developer or to any third-party server.
  • It does not collect, store, or sell user data.
  • It uses no analytics and no tracking.
  • It has no backend; all processing happens locally in the browser.

Permissions

cookies, scripting, and host access to oauth.telegram.org and app.send.tg are used solely for the function described above.
